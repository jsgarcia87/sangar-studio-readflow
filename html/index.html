Preguntas Frecuentes

Todo lo que necesitas saber sobre el plugin de audio Sangar Studio ReadFlow.

help_outline ¿Es obligatorio introducir una clave API de OpenAI?

expand_more

No. Si decides no ingresar una clave API u optar por mantener desactivada la voz de IA, el plugin funciona en modo fallback. Utiliza el motor nativo del navegador del usuario (mediante Web Speech API window.speechSynthesis) para narrar tus textos de forma totalmente instantánea y 100% gratuita, sin consumir cuota ni recursos en tu servidor.

help_outline ¿Dónde se almacenan y administran los archivos de audio?

expand_more

Los archivos MP3 generados se guardan de forma segura y permanente en la carpeta local "/wp-content/uploads/sangar-studio-readflow/" en tu propio servidor WordPress. Desde el panel de administración del plugin puedes visualizar el espacio total ocupado y vaciar la caché en cualquier momento con un solo clic.

help_outline ¿Cómo se invalida la caché de audio si modifico una entrada?

expand_more

ReadFlow es sumamente eficiente y automatizado. Cada vez que editas, actualizas o eliminas una entrada existente en WordPress, el plugin detecta el evento y elimina automáticamente el archivo MP3 correspondiente de la caché, garantizando que el próximo oyente escuche siempre la versión más reciente del post.

help_outline ¿El plugin soporta artículos extremadamente largos?

expand_more

¡Sí, sin límites! ReadFlow incorpora un algoritmo de división de texto inteligente. Segmenta los posts extensos en bloques semánticos de aproximadamente 3,500 caracteres, respetando oraciones completas y signos de puntuación (evitando romper palabras). Luego procesa cada segmento a través de la API y une el audio final en un solo MP3 fluido, superando la barrera de 4,096 caracteres de OpenAI de forma transparente.

help_outline ¿Cómo puedo insertar el reproductor manualmente en mi tema o post?

expand_more

Por defecto, el plugin puede inyectar de forma automática el widget al inicio o al final del contenido del post. No obstante, puedes desactivar la inyección automática en ajustes y usar el shortcode "[sangar-studio-readflow]" (o el alias clásico "[readio]") en cualquier bloque de Gutenberg, Elementor, Divi o llamarlo programásticamente en tu plantilla PHP.
